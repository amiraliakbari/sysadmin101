About the Course

Linux server administration course aims to provide a comprehensive introduction to linux ops and server management techniques. Programmers will be able to professionally maintain servers of their projects using concepts learned in this course.

Course Organizers

  • AmirAli Akbari

    The course is compiled and teached by AmirAli Akbari, who has several years of experience in managing production servers for some high load websites.

  • Arsh Company

    The initial planning and first series of this course is sponsored by Arsh company.

Lessons List

    Topics: SSH Security, Log Analysis, Permissions, Sudo, Security Utilities, Secure Deployment Architecture, Common Web Attacks and Defences.

Interesting Topics

  • From VMs to Containers

    Virtualization, VMWare ESXi, Chroots & cgroups, Linux containers, Docker, Container Orchestration (kubernetes, Docker Swarm)

  • Configuration Management

    Automated server setup, Phoenix server, Puppet, Ansible
